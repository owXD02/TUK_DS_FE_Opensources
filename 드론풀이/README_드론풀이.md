# 드론 주제 예상 문제 풀이 (드론 1~20)

`드론예상문제.txt`의 20문제를 **DS_LT_Opensources 라이브러리 조립 위주**로 푼 답안.
20개 전부 VS2019에서 **경고/에러 0개로 컴파일 검증** 완료.

## 컴파일 방법
1. VS 빈 프로젝트 생성
2. 풀려는 `드론NN_*.c` 1개 + **DS_LT_Opensources의 .h/.c 전부** 추가
   (라이브러리 `main.c`는 main 중복이므로 **프로젝트에서 제외**)
3. 빌드/실행

- 그 외: drone_path.txt / drone_list.txt 등 자동 생성

## 핵심 포인트
- 경로(좌표) 문제는 **xyList**(양방향·FIFO)와 **XYStack**(역순)으로 거의 해결.
- 편대 명단(id/좌표/배터리) 문제는 **drone**의 정렬/검색 함수로 해결.
- "배터리 적은/많은 순", "우선순위"는 **heap + tag**(키=값, tag=id)로 해결.
- 직접 구현이 필요한 건 거리 계산·영역 판정·배터리 시뮬 같은 '문제 고유 로직'뿐.
