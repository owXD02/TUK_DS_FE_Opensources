# 예상문제 풀이 모음 (문제 1~20)

`예상문제.txt`의 20문제를 **DS_LT_Opensources 라이브러리 조립 위주**로 푼 답안.
20개 전부 VS2019에서 **경고/에러 0개로 컴파일 검증** 완료.

## 컴파일 방법
1. VS 빈 프로젝트 생성
2. 풀려는 문제의 `.c` 1개 + **DS_LT_Opensources의 .h/.c 전부** 추가
   (단, 라이브러리의 `main.c`는 main 중복이 되므로 **프로젝트에서 제외**)
3. 빌드/실행. (입력 파일이 필요한 문제는 아래 표 참고)

## 입력 파일이 필요한 문제 (없으면 프로그램이 안내/자동생성)
- 문제 6 : `postfix.txt` (예: 한 줄에 `52+8*`)
- 문제 8 : `words.txt` (영어 단어들, 공백/줄 구분)
- 문제 18 : `inventory.txt` (없으면 자동 생성)
- 문제 20 : `tasks.txt` (없으면 자동 생성)
- 그 외 문제는 난수를 직접 생성하므로 입력 파일 불필요.

## "직접 구현"이 많이 들어간 3가지 (라이브러리로 조립 불가했던 부분)
- **문제 8 문자열 BST** : 라이브러리 bst는 int 키 전용 → 문자열 키 BST를 직접 작성(주석 상세).
- **문제 16 기수 정렬 단계 출력** : 라이브러리 radixSort는 중간 단계를 안 보여줌 → linkedQueue를 버킷으로 써서 자리별 출력하도록 직접 작성.
- **문제 19 그래프 BFS** : 그래프는 라이브러리 범위(큐~정렬) 밖 → 인접행렬+BFS를 직접 작성(단, BFS 큐는 linkedQueue 재사용).

이 세 가지를 제외하면 대부분 라이브러리 함수 호출 + 문제 고유 로직(주석 처리)만으로 구성됩니다.
