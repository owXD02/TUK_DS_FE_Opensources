=====================================================================
            드론(Drone) 주제 자료구조 기말 예상 문제 20선
=====================================================================
범위 : 큐 ~ 정렬. 모두 DS_LT_Opensources 라이브러리 조립으로 풀이.
풀이 : 드론풀이/ 폴더 (드론01_*.c ~ 드론20_*.c)
사용 핵심 라이브러리 : drone, xyList, stack, deque, cQueue, linkedQueue,
                       heap, bst, avl, sort, utils

[큐/스택/덱 - 경로]
 1. 순찰 경로 생성 + 왕복 시뮬레이션 (createDronePathFile + patrolSimulation)
 2. 자폭 임무: 경로를 역순으로 비행 (좌표 스택)
 3. 경로를 연결 큐(FIFO)에 넣고 순서대로 비행 (xyList)
 4. 정찰 후 복귀: 경로 양방향 순회 (xyList 전진/복귀)
 5. 배터리 소모 시뮬레이션 + 추락 지점 찾기 (직접 시뮬)

[정렬/탐색 - 편대 명단]
 6. 드론 N대 명단 생성→파일→배터리 내림차순 정렬 (sortDronesBy)
 7. 배터리 가장 적은 드론부터 충전 (최소 힙)
 8. 배터리 상위 K대 선발 (최대 힙)
 9. id로 드론 검색: 순차 vs 이진 탐색 비교
10. 드론 id를 BST에 등록하고 검색 (bst)
11. x좌표 / y좌표 기준 정렬 (sortDronesBy)
12. 특정 사각 영역 안의 드론 개수 세기 (필터)
13. 두 드론의 최단/최장 거리 쌍 찾기 (거리 계산)

[힙/트리/시뮬 - 운용]
14. 임무 우선순위 스케줄링 (최대 힙)
15. 경로 총 비행 거리 계산 (맨해튼 거리 합)
16. 저배터리 경고 필터 + 파일 기록 (임계값)
17. 드론 x좌표를 선택/삽입/버블 정렬로 단계 출력
18. 드론 착륙 대기열 (원형 큐)
19. 드론 id 순차 등록 시 AVL로 균형 유지 (vs 일반 BST)
20. 종합 관제 시스템 (생성/정렬/검색/충전/순찰 메뉴 통합)

※ 거의 모든 문제가 "난수 생성 → 파일 저장/읽기 → 자료구조 처리 →
   화면(+파일) 출력" 흐름. 직접 구현한 부분은 각 .c 파일에 상세 주석.
=====================================================================
